    Escrever conteúdo em arquivo

    echo "Hello, World!" > hello.txt

    Listar arquivos

    ls -lha
    • l: visualizar como lista
    • h: legível para humanos
    • a: mostrar arquivos ocultos

    Mostrar documentação do comando

    man ls

    Exibe o conteúdo de um arquivo

    cat hello.txt

    Exibe as primeiras linhas de um arquivo

    head -n 10 /proc/cpfuinfo
    • n: Número de linhas

    Exibe as últimas linhas de um arquivo

    tail -n 10 /proc/cpfuinfo
    • n: Número de linhas

    Exibe conteúdo de arquivo com navegação

    less /var/log/messages

    busca termos dentro de arquivos ou diretórios

    sudo grep gabriel /var/log/auth.log

    Copiar arquivo

    To Copy

    cp hello.txt teste.txt

    Renomear ou mover arquivo

    To Move

    mv hello.txt hello.bak

    Permissão pare executar

    Change Mode

    chmod +x <nome_arquivo>

    Onde x faz referência ao x de e## x## ecute

    Descompactar arquivos zip

    unzip -tv <nome_arquivo>.zip
    • t: testa o arquivo antes de descompactar.
    • v: executa o comando de maneira verbosa.

    (Des)empacota ou (des)comprime arquivos

    tar -czvf arquivos.tgz *.txt
    • c: indica que o arquivo está sendo criado.
    • z: indica que o arquivo será comprimido com gzip.
    • v: executa o comando de maneira verbosa.
    • f: nome do arquivo que será gerado.
